发表于 2019-12-20 更新于 2019-12-25 分类于 hexo

1.1生成SSH-KEY

在~/.ssh/目录会生成github-rsa和github-rsa.pub私钥和公钥。 我们将github-rsa.pub中的内容全部粘帖到github服务器的（Settings –> Deploy keys –> Add deploy key）SSH-key的配置中。

1.2测试配置是否成功

成功提示：Hi sweep-monk/blog! You’ve successfully authenticated, but GitHub does not provide shell access.
